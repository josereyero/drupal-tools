Drupal Tools
=============

Introduction
------------
This is a library to be used by some Drupal development tools.

By https://reyero.net
Use at your own risk.

Requirements
------------
* Drupal 8.x, https://drupal.org
* Drush 9.x, http://docs.drush.org/en/9.x/
* Ansible, https://www.ansible.com/

Installation
------------

* Via composer:
  composer require reyero/drupal-tools


Projects using Drupal Tools
----------------------------

* Project Tools 8.x, https://github.com/josereyero/project_tools
* Site Tools 8.x, https://github.com/josereyero/site_tools
